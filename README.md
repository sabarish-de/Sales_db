# Sales Data Analysis

A summary of order records and monthly revenue performance across 2023 and 2024, based on SQL query results from the sales database.

## Overview

This project analyzes sales transactions for three products (101, 102, 103) covering January 2023 through mid-2024.

## Data Fields

- `order_id` — Unique order identifier
- `order_date` — Date of the transaction
- `amount` — Order value in USD
- `product_id` — Product identifier (101, 102, or 103)
